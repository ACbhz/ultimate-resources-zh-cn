### 简介

Uninstall Tool 是一款功能强大的专业卸载工具。它比 Windows 自带的“添加/删除程序”功能快 3 倍！它可移除系统自带程序功能所不能删除的程序，使用强制删除可卸载注册表相关项，支持显示隐藏的安装程序，按名称过滤已安装程序列表，强行卸载程序，浏览注册表项目，保存安装程序列表，软件小巧，操作简便。
